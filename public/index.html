使用方法

这个工具用来把 Postype 韩文小说正文翻译成简体中文。

是朕执意要看的，要罚就罚朕吧。

输入方式
  1. 输入 Postype 页面 URL
  2. 手动复制粘贴韩文正文
  3. 上传 .html 或 .webarchive 文件
付费和成人内容不支持 URL 输入，需要手动复制粘贴文本，或者导入文件。
注意事项
  • 只适配Postype。
  • .webarchive 文件会解析主资源中的 HTML 内容。但因为文件一般太大，所以基本是摆设状态。
  • 翻译质量和速度会受到不同因素的影响。如果翻译不理想，建议朴素地重试一次。
