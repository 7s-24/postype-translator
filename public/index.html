快速模式说明

快速模式会并行翻译多个段落，通常速度更快。

非快速模式会按段落顺序逐段翻译，上下文衔接通常更稳，但耗时更长，也更容易受到普通额度限制。

如果非快速模式额度不足、请求失败或等待太久，可以切换快速模式继续尝试。

快速模式的代价是段落之间可用上下文较少，长篇剧情、称呼变化或连续对话可能需要人工检查。

使用方法

这个工具用来把 Postype 韩文小说正文翻译成简体中文。

是朕执意要看的，要罚就罚朕吧！

输入方式
  1. 输入 Postype 页面 URL
  2. 手动复制粘贴韩文正文
  3. 上传 .html 或 .webarchive 文件
付费和成人内容不支持 URL 输入，需要手动复制粘贴文本，或者导入文件。
注意事项
  • 只适配Postype。
  • .webarchive 文件会解析主资源中的 HTML 内容。但因为文件一般太大，所以基本是摆设状态。
  • 翻译质量和速度会受到不同因素的影响。如果翻译不理想，建议朴素地重试一次。
