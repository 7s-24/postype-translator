# 术语库说明

术语库用于固定韩文词条的中文译法。

## 字段

每条术语包含三个字段：

- ko：韩文原文
- zh：中文译法
- category：类别

## 类别

可用类别：

- 人名
- 地名
- 技能
- 称号
- 物品
- 组织
- 称呼
- 艺名
- 本名
- 其他

## 示例

```json
[
  {
    "ko": "신유",
    "zh": "申惟",
    "category": "艺名"
  }
]
```

## 默认术语库

默认术语库文件位于：

```txt
public/data/glossary.default.json
```

页面初始化时会优先读取该文件。

如果读取失败，会自动回退到内置默认术语库。

## 浏览器存储

全局术语库保存在浏览器 localStorage。

清除浏览器数据可能会删除术语库。

## 导入导出

页面支持：

- 导出术语库 JSON
- 导入符合格式的 JSON 文件

## 优先级

全局术语库优先级最高，不会被篇章术语提取覆盖。
